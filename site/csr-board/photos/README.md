# CSR photos

The ServiceTitan API does not expose employee profile photos, so the board
uses initials avatars until a photo is dropped in this folder.

Add one JPG per person, named `<company>-<employeeId>.jpg`:

    sierra-528566460.jpg
    ultimate-12345678.jpg
    russett-87654321.jpg

The employee id is in the `id` field of each row in `../data.json`
(or the ServiceTitan URL when viewing the employee in Settings > People >
Employees). Portrait-ish crops look best; the card crops to fit automatically.
