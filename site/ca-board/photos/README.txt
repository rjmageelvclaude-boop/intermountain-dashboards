Comfort Advisor headshots go in this folder.

ServiceTitan's public API does not expose the technician bio photos, so export
them from ST (Settings > People > technician > bio picture) or use any photo,
and save each one here named either way:

  <servicetitan tech id>.jpg     e.g. 14593.jpg
  <name-slug>.jpg                e.g. jeremy-brackett.jpg  (lowercase, dashes)

.png works too. Anyone without a photo shows an initials tile instead.
